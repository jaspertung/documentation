Position Property

Static

positioned according to normal flow of document (default), t, r, b, l, and z-index properties have no effect

Relative

positioned according to normal flow, then offset relative to itself based on t, r, b, l values

Absolute

not positioned according to normal flow, relative to closest positioned ancestor if none, then to initial containing block (in this case- body)

Fixed

positioned relative to intial containing block- stay even while scrolling (nav bar)

Sticky

positioned according to normal flow, then offset relative to nearest scrolling ancestor and containing block (nav bar) how do?
